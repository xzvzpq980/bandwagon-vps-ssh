## 登录 VPS 需要哪三样东西

先说个概念。**搬瓦工使用 IP 登录 VPS**，本质上是通过 SSH 协议远程连接到服务器。这跟你平时用桌面软件不一样，需要三个必备信息：

1. **IP 地址**（Public IP address）：你的 VPS 的网络地址，相当于门牌号
2. **SSH 端口号**（SSH Port）：搬瓦工的端口不是默认的 22，是随机生成的五位数，必须手动查
3. **root 密码**：登录用的密码，搬瓦工现在不会自动发，必须手动重置一次

用户名不用找，默认就是 `root`，小写，固定的。

三样东西都找到了，才能连。少任何一样，都连不上。

## 第一步：去 KiwiVM 面板找 IP 和端口

搬瓦工有两个后台：一个是账户后台（My Services 页面），一个是 KiwiVM 控制面板。找 IP 要去 KiwiVM 这里。

**操作路径如下：**

1. 登录搬瓦工账户，进入「My Services」页面，找到你买的 VPS
2. 点击右边「KiwiVM Control Panel」按钮，进入这台 VPS 的 KiwiVM 面板
3. 进入后，面板首页就能看到两行关键信息：
   - **Public IP address**：这就是你的 IP 地址
   - **SSH Port**：这就是端口号，五位数左右，不是 22

把这两个记下来，或者直接截个图留着用。

## 第二步：重置 root 密码（只显示一次，记好）

这是新手最常踩的坑——以为密码会在邮件里，其实搬瓦工出于安全考虑，不存储也不明文发送当前的 root 密码。想用，就得手动重置一次。

**重置步骤：**

1. 先确认 VPS 是**开机状态**（这个坑很隐蔽：密码重置需要 VPS 在线，关机状态下会报错 739102）
2. 在 KiwiVM 面板左侧菜单，找到「Root password modification」
3. 点击「Generate and set new root password」
4. 页面顶部会出现绿色背景的新密码

**重要提醒**：这个密码只出现这一次。页面刷新或者关掉就看不到了。必须马上复制保存好，放在能记住的地方。

忘了怎么办？再走一遍重置流程，重新生成一个就行。

## 第三步：选 SSH 工具连接 VPS

有了 IP、端口、密码，下一步就是用 SSH 软件连过去。不同系统用法稍有差异：

### Windows 用户

Windows 没有内置好用的 SSH 图形界面工具，需要另外下载。目前用得最多的是这两个：

- **Termius**：跨平台，界面简洁，支持 Windows/Mac/iOS/Android，有免费版，目前推荐首选
- **Xshell**：老牌工具，免费个人版够用，界面更传统

以 Xshell 为例，连接步骤大致如下：

1. 新建会话，「主机」填入 IP 地址
2. 「端口号」填刚才记录的 SSH Port
3. 切到「用户身份验证」，方法选「Password」，用户名填 `root`，密码填刚才重置的那串
4. 点确定，然后连接，首次会弹出一个确认密钥的窗口，选「接受并保存」即可

连上之后能看到命令行提示符，就成功了。

### Mac 用户

Mac 自带终端，直接在终端里输命令就能连，不用装额外软件：

bash
ssh root@你的IP地址 -p 你的SSH端口号


回车，提示输密码时，把刚才保存的密码粘贴进去（注意：终端里输密码不会显示任何字符，这是正常的，输完直接回车）。

出现命令行提示符，就连上了。

### 用浏览器直接操作（应急用）

KiwiVM 面板左侧有个「Root shell - interactive」选项，相当于在浏览器里开了个终端窗口，不需要任何本地工具就能操作 VPS。这个适合临时用或者 SSH 突然连不上时应急，不建议长期靠这个操作。

## IP 被封了怎么办？

搬瓦工 VPS 的 IP 在国内有可能被封，主要表现是 SSH 突然连不上，ping 也没响应。

判断方式：用手机流量（不走家庭宽带）访问 VPS 的 IP，如果能通，说明是家庭宽带这段线路的问题；如果国内国外都 ping 不通，IP 可能真被封了。

遇到这种情况，可以在 KiwiVM 面板里申请更换 IP，目前换 IP 的费用是 $2.82。相比整套重建，价格还算合理。

## SSH 连不上时的常见排查思路

买完搬瓦工用 IP 登录时，遇到连不上的情况，按这个顺序检查：

**1. 确认 VPS 是开机状态**

进 KiwiVM 面板，看顶部状态是不是「Running」。刚买的 VPS 一般是开着的，但有时候会因为超流量或者账单问题自动关机。

**2. 确认三个信息都填对了**

IP 地址、端口号，各复制一遍，不要手打。端口号一定要用 KiwiVM 里的那个，不是 22。

**3. iptables 防火墙问题**

这是用 SSH 工具装过某些脚本之后经常出现的情况。规则写错了，自己把自己锁在门外。解决办法：用 KiwiVM 的「Root shell - interactive」功能进去，直接清一下 iptables 规则：`iptables -F`。

**4. VPS 处于超流量限速状态**

搬瓦工月流量用完之后，不会直接断网，但速度会被严重限制，SSH 超时很常见。在 KiwiVM 里可以看到当月流量使用情况，如果快用完了就等下个月重置，或者考虑临时扩充流量。

**5. 家庭宽带出口 IP 被封**

跟上面说的一样，换手机流量测一下，确认是本地网络问题还是 VPS IP 本身的问题。

步骤大致是：本地生成 SSH 密钥对，把公钥上传到 KiwiVM 的「Public SSH Keys」管理页面，后续重装系统时会自动配置好。实际用起来，连接直接跳过密码提示，顺畅很多。

这个功能对经常需要管理 VPS 的人来说很实用，KiwiVM 面板里有专门的管理入口，设置不复杂。

## 常见问题解答

**Q：搬瓦工的 SSH 端口默认是多少？**

搬瓦工的 SSH 端口是随机生成的，每台 VPS 都不一样，不是默认的 22。必须去 KiwiVM 面板首页看「SSH Port」那一行才能知道。

**Q：密码重置时提示 739102 报错怎么办？**

报这个错是因为 VPS 处于关机状态。先在 KiwiVM 面板把 VPS 开机（点「Start」），等状态变成「Running」之后，再去重置密码就正常了。

**Q：忘记密码了、又把页面关了怎么办？**

直接再重置一次就行。进 KiwiVM → Root password modification → Generate and set new root password，生成新密码，记好就行。

**Q：用 IP 登录后，VPS 里有什么系统？**

搬瓦工 VPS 出厂默认装的是 CentOS，也可以在 KiwiVM 面板里选其他系统重装，支持 Ubuntu、Debian、AlmaLinux、Rocky Linux 等二十几个版本，一键操作，重装完重新获取密码就行。

**Q：搬瓦工有退款吗？**
